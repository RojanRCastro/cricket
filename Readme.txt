1. Move the Bounce Marker
	* Use W / A / S / D to position the marker on the pitch.
	* This determines where the ball will hit the ground.
2. Start Bowling
	* Press Space (or Bowl button) once to start the swing/spin meter.
3. Set Power
	* Press Space again to stop the meter.
	* The closer the indicator is to the center, the stronger the effect.
4. Ball Delivery
	* The ball will be released automatically after stopping the meter.
	* It will travel toward the selected bounce point.
5. Toggle Bowling Type
	* Press Tab to switch between:
		* Swing Mode → Ball curves in the air before bounce
		* Spin Mode → Ball turns sharply at the bounce
